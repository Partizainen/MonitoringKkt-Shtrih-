# KKTMonitor - Система мониторинга контрольно-кассовых машин Штрих(теситровался только Штри01ф)

## Описание
KKTMonitor - это веб-приложение для централизованного мониторинга ККМ в розничной сети. 
Система позволяет автоматически обнаруживать ККМ, отслеживать их состояние и получать 
уведомления о критических событиях.

## Возможности
- 🔍 Автоматическое обнаружение ККМ через DHCP лизинг
- 📊 Мониторинг состояния ФН, смен, ошибок
- 📧 Email уведомления при изменении статуса
- 📈 Интеграция с Zabbix
- ⏰ Гибкое расписание опросов
- 🖥 Удобный веб-интерфейс

## Требования
- Windows Server 2016/2019/2022
- .NET 8.0 Runtime
- MySQL 8.0+
- DrvFR 5.25+
- IIS 10.0

## Установка
1. Установите DrvFR драйвер
2. Создайте базу данных MySQL
3. Выполните скрипт инициализации БД
4. Настройте appsettings.json
5. Опубликуйте проект: `dotnet publish -c Release`
6. Настройте сайт в IIS

## Конфигурация
- Порт по умолчанию: 5010
- Пароль доступа: MonitoringKkt
- Порт подключения к ККМ: 7778

## API Endpoints
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | /api/kkt | Список ККМ |
| POST | /api/kkt/poll/{id} | Ручной опрос |
